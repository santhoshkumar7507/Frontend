# Frontend: Software Development Lifecycle (SDLC) Outline

This project contains a simple HTML page that outlines the phases of the Software Development Lifecycle (SDLC).

## Files

- `Index.html`: A static HTML page containing an ordered list of SDLC phases and their sub-activities.

## Phases Covered

1. **Requirement Analysis**
   - Gather Business Requirements
   - Identify Stakeholders
   - Define Project Scope
2. **System Design**
   - Database Design
   - UI/UX Design
   - Architecture Planning
3. **Development**
   - Frontend Development
   - Backend Development
   - API Integration
4. **Testing**
   - Unit Testing
   - Integration Testing
   - User Acceptance Testing
5. **Deployment & Maintenance**
   - Production Release
   - Monitoring
   - Performance Optimization
